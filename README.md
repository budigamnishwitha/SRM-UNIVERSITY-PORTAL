🎓 SRM AP Portal (Frontend UI)

📌 Project Overview

This project is a frontend replica of a university portal designed using HTML and CSS. It provides a clean and structured interface for both freshers and senior students to access academic-related services.

## 🚀 Features

* Responsive login interface for students
* Separate sections for Freshers and Senior Students
* Clean and modern UI design
* Background image with glassmorphism effect
* Structured layout using Flexbox
* User-friendly form design

 🛠 Technologies Used

* HTML5
* CSS3 (Flexbox, Styling, Effects)

▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in any web browser
